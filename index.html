I began on a team assigned to external facing applications and eventually joined a new team dedicated to internal applications. Between these teams, I had the opportunity to work on projects written in JavaScript and C#. Some interesting technologies utilized by these projects include Node.js, AngularJS, ASP.NET, ServiceStack, and several others.

Kelp

The interpreter for a small functional programming language written in JavaScript. The inspiration for this project came from a programming languages course taught by Matthew Flatt at the University of Utah.

Commander Py

A Python module to provide a simple solution to complex command line applications. Based on Commander.js for Node.js, Commander Py allows users to process command line options and arguments with ultimate customizability.

JS-Expression

A Lisp/Racket style s-expression implementation for JavaScript. This allows for easy parsing of strings with tokens enclosed in parenthesis. I plan to use this implementation to update the Kelp language for more flexible parsing.

Super-Diff

A Node.js module built to find the differences between two chunks of text. Super-Diff was designed with the user in mind, giving the them the ability to tailor the output to their specific needs.

Minimal Prompt

A Node.js module providing an API for gathering information from a user via the command line. Designed for simplicity, this module allows the user to ask a specific set of questions, or repeatedly prompt the user for information.
